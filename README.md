# Pantone Wallet — Premium Currency Converter

Кроссплатформенное приложение в виде раскрывающегося премиального кошелька. Конвертирует валюты по актуальному курсу. Работает как PWA (на компьютере и телефоне) и как Telegram Mini App.

## Что особенного

### Уникальная система Pantone — каждый месяц новые цвета

Кошелёк раскрашен в официальные цвета года Pantone. В базе **все 27 цветов с 2000 по 2026 год**.

**Правила цветовой смены:**

1. **До мая 2026** — классическая пара текущего года:
   - Обложка: **Mocha Mousse 2025** (#A47864)
   - Внутри: **Cloud Dancer 2026** (#F0F0F0)

2. **С мая 2026** — каждый месяц новая пара из 27 цветов Pantone, по строгим правилам:
   - **Обновление ровно раз в месяц** — цвета меняются 1-го числа каждого месяца
   - **Внутренняя часть кошелька всегда светлее обложки** — алгоритм автоматически ставит более светлый цвет внутрь, более тёмный — снаружи
   - **Минимальная разница яркости — 40 единиц** — если случайно выбраны слишком похожие цвета, алгоритм ищет следующий контрастный
   - **Минимум повторений** — формула с взаимно простыми коэффициентами (31 и 17) обеспечивает разнообразие пар на годы вперёд

**Результат:** Каждый месяц кошелёк выглядит свежо, но всегда сохраняет премиальный контраст между обложкой и внутренней частью.

### Анимация раскрытия
Реалистичная анимация открывающегося кошелька — обложка отъезжает вправо, внутренняя часть с конвертером выезжает влево. Плавная, как настоящая кожаная книжка.

### Контрастность текста — автоматическая
В зависимости от яркости фона цвет шрифта (чёрный или белый) подбирается автоматически — текст всегда читается на 100%.

### Золотые элементы
Слитки на обложке нарисованы CSS-кодом — без картинок, поэтому масштабируются на любой экран без потери качества.

### Иконка приложения
Уникальная SVG-иконка в стиле приложения: тёмный фон + терракотовая обложка кошелька + три золотых слитка лесенкой (символ роста). Векторная — чёткая на любом экране, в том числе 4K и Retina.

## Алгоритм цветов — как это работает в коде

В `script.js` функция `pickColorPair(years, monthIndex)`:

1. Вычисляет индекс месяца с мая 2026: `monthIndex = (год - 2026) * 12 + (месяц - 4)`
2. Из 27 цветов выбирает два индекса по формуле: `(monthIndex * 31 + 5) % 27` и `(monthIndex * 17 + 11) % 27`
3. Сравнивает яркость через формулу YIQ: `(R*299 + G*587 + B*114) / 1000`
4. Если разница яркости меньше 40 — берёт следующий цвет, пока не найдёт контрастную пару
5. Возвращает: `{ body: светлый_hex, cover: тёмный_hex }`

Логика гарантирует:
- Внутренняя часть **всегда светлее**
- Контраст всегда достаточный для читаемости
- Пары не повторяются часто благодаря взаимно простым коэффициентам

## Как использовать

### В браузере
Открой ссылку. Конвертер сразу готов к работе.

- **Клик по обложке** — кошелёк раскрывается
- **Введи сумму** в любом из двух полей — второе пересчитается автоматически
- **Кнопки RUB/USD/EUR** или выпадающий список — для выбора валют
- **Иконка $€¥₽** в правом верхнем углу — переход на полный конвертер РБК
- **Никнейм @AllaShenni** внизу — Telegram автора

### Как установить на компьютер (PWA)
1. Открой ссылку в Chrome или Edge
2. В адресной строке справа появится иконка установки (или золотая кнопка «Скачать на Рабочий стол» внутри приложения)
3. Нажми — приложение установится как обычная программа
4. Запуск двойным кликом — открывается в отдельном окне без браузерной строки

### Как установить на телефон
- **iPhone:** Открой в Safari → кнопка «Поделиться» → «На экран Домой»
- **Android:** Открой в Chrome → меню → «Установить приложение» или «Добавить на главный экран»

После установки иконка кошелька появляется на главном экране — запускается как обычное приложение.

### В Telegram (Mini App)
Если приложение подключено к Telegram-боту через `@BotFather`, при открытии в Telegram развернётся на весь экран автоматически.

### GitHub Pages (бесплатно)
1. Залить все файлы в репозиторий GitHub
2. Settings → Pages → Source: `main` branch, `/ (root)` → Save
3. Через 1-2 минуты приложение доступно по адресу `https://логин.github.io/имя-репо/`
4. PWA-установка работает только по HTTPS — GitHub Pages даёт его автоматически

### Telegram Mini App
1. Написать [@BotFather](https://t.me/BotFather) в Telegram → `/newbot` → создать бота
2. `/setmenubutton` → выбрать бота → ввести URL приложения
3. Бот получает кнопку запуска приложения

## Курсы валют

Используется бесплатный публичный API: **exchangerate-api.com**.

- Курсы кэшируются на стороне сервиса, обновляются ежедневно
- Если интернета нет, конвертер показывает «Нет связи с курсами»
- Никаких API-ключей не требуется
