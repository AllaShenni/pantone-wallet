# Техническое задание: Валютный кошелек "Pantone Wallet"

## Описание проекта
Кроссплатформенное приложение (PWA), представляющее собой конвертер валют в стилистике раскрывающегося кошелька. Цвет кошелька динамически обновляется в соответствии с цветом года Pantone.

## Основные функции
1.  **Динамический дизайн**:
    - Цвет года подтягивается автоматически (через API или парсинг).
    - Анимация раскрытия кошелька при нажатии.
2.  **Конвертация валют**:
    - Основные валюты: RUB, USD, EUR.
    - Возможность добавления пользовательской валюты.
    - Автоматическое определение локальной валюты по IP/геолокации.
3.  **Интеграция**:
    - Переход на конвертер РБК.
    - Курсы в реальном времени.
4.  **Авторство**:
    - Элегантная подпись разработчика.

## Технологический стек
- **Frontend**: HTML5, CSS3 (Flexbox/Grid, Animations), JavaScript (Vanilla).
- **Формат**: Progressive Web App (PWA) для работы на PC, Android и iOS.
- **API**:
    - `ip-api.com` (Локация/IP).
    - `exchangerate-api.com` (Курсы валют).
    - Pantone color source (ТБД).

## План разработки
1.  **Шаг 1**: Создание базовой HTML-структуры и CSS-стилей "Кошелька".
2.  **Шаг 2**: Реализация анимации раскрытия.
3.  **Шаг 3**: Подключение API для определения местоположения и курсов валют.
4.  **Шаг 4**: Реализация логики получения цвета Pantone.
5.  **Шаг 5**: Финальная шлифовка UI и добавление ссылок (РБК).
